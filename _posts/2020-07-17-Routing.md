### 스위치와 라우터
---

| ------ | ------- | ------- |  
| 항목 | 스위치 | 라우터 |  
| 동작 계층 | L2 | L3 |   
| 용도 | 동일 서브넷 | 다른 서브넷 |  
| 참조 | MAC 테이블 | 라우팅 테이블 |  
| 유지 | Transparent bridging | Routing Protocol |  
| 루프 방지 | STP/RSTP/MSTP | 라우팅 프로토콜 |  
| 전송 | MAC 주소 | IP/IPv6 |  
| 브로드캐스트 패킷 | 플러딩 | 차단 |   
| 멀티캐스트 패킷 | 플러딩 | 차단 |   
| 목적지 모를때 | 플러딩 | 차단 |  
| 패킷 처리 속도 | 빠름 | 느림 |
